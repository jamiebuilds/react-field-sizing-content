# react-field-sizing-content

> A React-based ponyfill for `field-sizing: content`

## Install

```sh
npm add react-field-sizing-content
```

## Usage

```tsx
import { Input } from "react-field-sizing-content"

render(
	<Input
		fieldSizing="content"
		// otherwise is identical to `<input/>`
	/>
)
```
